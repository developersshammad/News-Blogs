# News & Blogs

A modern, responsive news and blogs website featuring articles on technology, sports, health, and travel. Built with HTML, CSS, Bootstrap, and JavaScript.

## Features

- Responsive design with Bootstrap
- Article categories (Technology, Sports, Health, Travel)
- Search functionality
- Article preview modals
- Pagination for article navigation
- SEO optimized with meta tags, Open Graph, and JSON-LD structured data
- Comments system (frontend only)

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (jQuery)
- Font Awesome icons

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Navigate through different categories and articles

## File Structure

- `index.html` - Homepage with latest articles
- `category.html` - Category-specific article listings
- `article.html` - Individual article pages
- `script.js` - Main JavaScript functionality
- `styles.css` - Custom styles
- `robots.txt` - SEO robots file
- `sitemap.xml` - XML sitemap for search engines

## SEO Features

- Dynamic meta tags for articles and categories
- Open Graph and Twitter Card support
- JSON-LD structured data
- Canonical URLs
- Robots.txt and sitemap.xml
